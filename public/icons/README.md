# Icons PWA

Placer ici les icônes pour le manifest :
- `icon-192.png` (192x192 px)
- `icon-512.png` (512x512 px)

Outil recommandé pour générer toutes les tailles : https://realfavicongenerator.net/

Design suggéré pour le MVP : cercle vert brand (#16a34a) avec une raquette stylisée blanche au centre.
